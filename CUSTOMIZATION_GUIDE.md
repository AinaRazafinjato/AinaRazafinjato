# üìù Customization Guide for Your GitHub Profile README

This guide will help you personalize your GitHub profile README with your own information.

## üéØ Quick Start - What to Customize

### 1. **Typing Animation Header** (Line 5)
Replace the text in the typing animation URL:
- Current: `Hi%2C+I'm+Aina+Razafinjato!`
- Change to: Your name with `%2C` for commas and `+` for spaces
- Example: `Hi%2C+I'm+John+Doe!`

### 2. **About Me Section** (Lines 17-24)
Update these placeholders with your information:
- **Currently working on**: Your current projects or role
- **Learning**: Technologies you're currently learning
- **Collaboration**: What kind of projects you want to work on
- **Ask me about**: Your areas of expertise
- **Fun fact**: Something interesting about you

### 3. **Tech Stack & Skills** (Lines 30-63)
**Languages**: Add/remove badges based on your skills
- Find more badges at: https://shields.io/ or https://github.com/Ileriayo/markdown-badges

Example badge format:
```markdown
![BadgeName](https://img.shields.io/badge/NAME-COLOR?style=for-the-badge&logo=LOGO&logoColor=white)
```

**Frameworks**: Update with frameworks you actually use
**Databases & Tools**: Customize based on your tech stack

### 4. **Featured Projects** (Lines 113-116)
Replace `project1`, `project2`, `project3`, `project4` with your actual repository names:
```markdown
[![ReadMe Card](https://github-readme-stats.vercel.app/api/pin/?username=AinaRazafinjato&repo=YOUR_REPO_NAME&theme=radical&hide_border=true&bg_color=0D1117)](https://github.com/AinaRazafinjato/YOUR_REPO_NAME)
```

### 5. **Work Experience & Education** (Lines 133-146)
Fill in your professional background:
- **Position**: Your job title
- **Company**: Your company name
- **Duration**: Start and end dates
- **Key Achievements**: Your accomplishments
- **Degree**: Your educational qualification
- **Institution**: Your school/university
- **Year**: Graduation year

### 6. **Social Links** (Lines 156-163)
Update all social media links:
- **LinkedIn**: `https://linkedin.com/in/YOUR-PROFILE`
- **Twitter**: `https://twitter.com/YOUR-HANDLE`
- **Portfolio**: `https://YOUR-PORTFOLIO.com`
- **Email**: `mailto:YOUR-EMAIL@example.com`
- **Dev.to**: `https://dev.to/YOUR-USERNAME`
- **Medium**: `https://medium.com/@YOUR-USERNAME`

### 7. **Spotify (Optional)** (Line 185)
To add Spotify now playing widget:
1. Visit: https://github.com/novatorem/novatorem
2. Follow setup instructions
3. Replace the URL in the README

## üé® Color Themes

The README uses a "radical" theme with custom colors:
- Background: `#0D1117`
- Title: `#F85D7F`
- Icons: `#F8D866`

To change themes, replace `theme=radical` with other options:
- `dark`, `tokyonight`, `onedark`, `cobalt`, `synthwave`, `highcontrast`, `dracula`

## üêç Snake Animation Setup

The snake animation will automatically generate when:
1. You push to the main branch
2. Every 12 hours automatically
3. You manually trigger it from Actions tab

**Note**: The first time it runs, it will create an `output` branch with the generated SVG files.
